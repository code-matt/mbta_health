## Installation
!! You need an API key from the MBTA in order to seed this project.
Place it inside .env ```MBTA_KEY=<your key>```
```
cd MBTA_Health
bundle install
rails db:create
rails db:migrate
rails db:seed
```
```
cd client
npm install
ng build
rails s
```
